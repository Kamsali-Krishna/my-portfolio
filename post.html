On Road Service Management System

This was a project I started back in july 2020 when I first learned python. I was the sole creator/developer for the first year of the project before I was able to hire help.

The system stored all the data of near by mechanics and send data to clients.

With the help of this data user/clients will know how are there in near by place and this database also provides the contact number of the mechanics so that user can contact directly and solve the issues.

Customers had the ability to view all their data with a dynamic search form, trend data, and physical location.

I worked on this system for 1 month before making it live.

Technologies:
  • - Python
  • - Django + Django REST Framework
  • - React JS
  • - Postgres
  • - AWS (S3 & RDS)
  • - Zingcharts - Trending & graphing test results
  • - Google Maps API
  • - HTML5 CSS3 JS
